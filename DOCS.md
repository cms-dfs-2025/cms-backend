## .env

Перед запуском проекта нужно создать файл `.env`, в котором нужно определить
переменные
```bash
# Пароль для подключения к базе данных
POSTGRES_PASSWORD=

# Пользователь для подключения к базе данных
POSTGRES_USER=

# Название sql базы данных
POSTGRES_DBNAME=

# Локальный порт для sql бд
POSTGRES_CONNECT_PORT=

# Локальный порт для сервера
SERVER_PORT=
```
